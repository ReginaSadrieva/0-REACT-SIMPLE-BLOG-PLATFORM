## ᚱ Running the Project Locally

```bash
1. Clone the repository
git clone https://github.com/ReginaSadrieva/0-REACT-SIMPLE-BLOG-PLATFORM.git

2. Install dependencies
npm install

3. Start the development server
npm run dev

4. Open in browser
http://localhost:5173

```
